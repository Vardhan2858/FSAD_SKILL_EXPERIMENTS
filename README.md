# FSAD Skill Experiments

This repository contains Full Stack Application Development (FSAD) laboratory experiments implemented using Git, Java, Maven, and Spring Framework.

## Technologies Used

* Git
* GitHub
* Java
* Spring Framework / Spring Boot
* Maven
* Eclipse / Spring Tool Suite (STS)

## Experiments

### SKILL – 1 : Git Basics

This experiment demonstrates basic Git commands used for version control.

Topics Covered:

* Git initialization
* Adding files to staging area
* Commit changes
* Checking repository status
* Viewing commit history

---

### SKILL – 2 : GitHub Integration

This experiment demonstrates connecting a local Git repository with GitHub.

Topics Covered:

* Creating GitHub repository
* Adding remote repository
* Pushing project to GitHub
* Pulling updates from GitHub

---

### SKILL – 3 : Spring Project Setup

This experiment demonstrates creating a Spring Maven project and running a basic Spring application.

Topics Covered:

* Maven project creation
* Spring dependency configuration
* Running a basic Spring program

---

### SKILL – 4 : Spring Dependency Injection

This experiment demonstrates Spring Dependency Injection using constructor and setter injection.

Topics Covered:

* IoC Container
* Constructor Injection
* Setter Injection
* XML configuration (applicationContext.xml)

---

### SKILL – 5 : Spring Autowiring

This experiment demonstrates Spring Autowiring using annotations.

Topics Covered:

* @Component
* @Autowired
* Component scanning
* Annotation-based configuration

---

## How to Run Spring Projects

1. Clone the repository

git clone https://github.com/Ujwal200707/FSAD_SKILL_EXPERIMENTS.git

2. Open the project in Eclipse / STS

3. Update Maven dependencies

Right Click Project → Maven → Update Project

4. Run the application

Right Click App.java → Run As → Java Application
